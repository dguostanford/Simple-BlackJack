A blackjack game in the terminal

![](_static/game_screenshot.png)

## Installation

```
$ git clone https://github.com/dguostanford/Simple-Blackjack.git
$ cd blackjack
$ pipenv install
```

## Play the game

```
$ 1. pipenv shell
$ 2. python -m blackjack
```
